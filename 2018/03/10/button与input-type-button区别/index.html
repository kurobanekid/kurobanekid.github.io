button与input type="button"区别

button控件是按钮，可以看做是一个容器，提供了展现表现力更好内容（文本或图像等多媒体内容）的可能，就如div那样可以“存放” < img > < p > < h1 > 等等的元素。
而< input type=”button” />也是按钮，功能上与button没有区别，表现力相对来说没那么强。如果需要表现图像，可以将input type设置为image，但是点击它时（被包含在表单中），它会提交当前表单，而input type=”button”，被点击后并不会提交表单，默认不做任何事情。

注意事项

  1. IE下，button默认type=”button”；非IE下，button默认type=”submit”

  2. 根据第一点，可以知道，当button被包含在form表单时（非IE情况），默认的type=”submit”，也就说点击它会提交所属的表单。功能上，相当于< input type=”submit” />

  3. 在表单中，IE下,button获取value将会返回”innerText”；非IE下，button获取value将会返回”attrValue”，因此建议在表单中使用input来创建按钮
