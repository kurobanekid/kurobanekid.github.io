postion属性值

  • static(默认)
  • inherit
  • relative
  • absolute
  • fixed

static

不指定position，默认为static，忽略top、 bottom、left、right或z-index声明

inherit

从父元素继承position值

接下来主要介绍后三个属性

relative(相对定位)

主要特点：

  1. 不脱离文档流
  2. 相对自身位置进行偏移

通过图片可以看到，相对定位并没有影响其他元素的位置、大小。

absolute(绝对定位)

主要特点：

  1. 脱离文档流
  2. 元素宽度变为内容宽度（包裹内容）a
  3. 根据最近的一个定位（relative/absolute/fixed）祖先元素进行偏移
  4. 偏移的参考区域为目标元素（上一点↑）的padding-box(內边距区)

存在已定位的祖先元素

首先可以看到目标元素已经脱离文档流，对父元素造成了破坏；
其次可以清楚看到目标元素的宽度已经变为内容宽度；
最后由于存在定位的祖先元素，因此以最近一个定位祖先元素(.container)进行偏移；

没有定位的祖先元素

如果祖先元素都没有设置定位，那么就以视窗为参照物，进行偏移。其他与上一种情况一样。

fixed

主要特点：

  1. 脱离文档流
  2. 元素宽度变为内容宽度（包裹内容）
  3. 根据视窗(window)进行偏移

可以看到它和absolute没有定位祖先元素的情况一样。
